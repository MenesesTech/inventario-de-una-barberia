# Sistema de Control de Inventario de productos

Ofrece seguimiento en tiempo real, registro de productos y notificaciones de bajo stock.

Diseñado para gestionar eficientemente el inventario de la barbería.

# Requerimientos

** Java

** Mysql

** NetBeans IDE
